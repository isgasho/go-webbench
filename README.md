# gowebbench - refine webbench in C

this software is licence under GPLv3

## Usage
```
./main [-c clientNumber -t requestTimeInSecond --(httpMethod)] url
```

## Show Help
```
./main
```

## How To build
```
git clone https://github.com/qinyuhang/go-webbench.git
cd go-webbench
make
```
